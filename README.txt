# Etiquetas-automaticas-Apli-10199

Generador automático de etiquetas HPLC (APLI 10199).

Cómo ejecutar localmente:
1. Crear un entorno virtual: `python -m venv .venv`
2. Activar entorno y instalar dependencias: `pip install -r requirements.txt`
3. Ejecutar: `streamlit run streamlit_etiquetador.py`

Archivo principal: `streamlit_etiquetador.py`
